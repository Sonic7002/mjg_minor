# Todo List Application (CLI)

This is a command-line based Todo List application written in Python.
The project focuses on core programming concepts, clean logic, and maintainability rather than UI or frameworks.

The application allows a user to manage daily tasks efficiently in an offline environment.
---
## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using a Queue
- Fully offline and lightweight
- Clear separation between logic, storage, and user interaction

## The project is organized into multiple files to improve readability and scalability:

### main.py
- Handles user interaction, menu display, and program flow.

### tasks.py
- Contains the core logic for managing tasks (add, remove, update, list).

### utils.py
- Contains helper functions such as screen clearing and pause utilities.

### README
- Project documentation.

## How to Run
- Clone the repository.
- Navigate to the project directory.
- Run the application using Python.
- Follow the on-screen menu options.

- Task description
- Error Handling

- Invalid menu input
- Invalid task IDs
- Empty task lists
- File read/write issues
